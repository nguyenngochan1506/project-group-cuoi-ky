Sử dụng Firebase để có thể đưa một ứng dụng lên Firebase

Bước 1: Truy cập trang web https://firebase.google.com

Bước 2: Đăng nhập hoặc đăng kí một tài khoản

Bước 3: Sau khi đăng nhập bấm vào Get started

Bước 4: Bấm vào Create a project

Bước 5: Điền project name và chọn hoặc tạo mới một tài khoản Google Analytics

Bước 6: Sau khi tạo xong, bấm vào phần realtime database

Bước 7: Chọn location cho database, chọn Start in test mode, chọn enable

Bước 8: Cấp quyền đọc ghi cho data base trong mục Rules sau đó chọn publish

Bước 9: Để kết nốt Firebase với app, chọn vào phần răng cưa, chọn project setting, sau đó bạn có thể chọn thêm Firebase vào app Android, ios, web hoặc Unity
